## Rick Cogley（コグレー・リック）

1987年から日本在住のバイリンガルITコンサルタント。横浜在住、東京で仕事。

### キャリア

1987年に東京大学の研究生として来日。動物実験よりプログラミングのほうがずっと面白いと気づき、IT業界へ。

最初の仕事（1989年頃）はテレカのデザイン会社。英語のキャプションやアイドル歌手の写真をライセンスしてテレカに印刷し、デパートやコンビニで販売する会社で、社内ネットワークの構築を担当しました。ちょうどIBM PS/2のリリース後に「DOS/V」が日本で普及し始めた頃です。PS/2が200万円した時代に、DOS/V機は25万円程度。そこから技術キャリアが始まりました。

その後、ネットワークサポート会社でヘルプデスク、ユーザーサポート、ネットワークサポートを経験。1993年にはPCトレーニング会社の共同設立に参加し、CIOとしてテクノロジーオペレーション全体の構築を担当しました。

1999年夏に[株式会社イソリア](https://esolia.co.jp)を創業。以来25年以上、東京で国際企業向けの日英バイリンガルITアウトソーシングとインフラサービスを提供しています。

### 現在の技術スタック

[SvelteKit](https://svelte.dev) + [Cloudflare Workers](https://developers.cloudflare.com/workers/)を中心に、TypeScriptでウェブアプリケーションを構築中。D1（SQLite）、R2、KVを活用したフルスタック構成です。

プロジェクト：
- [cogley.jp](https://cogley.jp)：技術、ビジネス、日本に関する記事（SvelteKit + Cloudflare Workers）
- [svelte.cogley.jp](https://svelte.cogley.jp)：React/Vue/AngularからSvelte 5への移行リファレンス（日英対応）
- [rick.cogley.jp](https://rick.cogley.jp)：プロフィールサイト
- [pulse.esolia.co.jp](https://pulse.esolia.co.jp)：セキュリティ・コンプライアンス管理。ISO 27001、CIS Controlsなどのフレームワークに基づく準拠追跡と証跡収集（SvelteKit + Cloudflare Workers）
- [periodic.esolia.co.jp](https://periodic.esolia.co.jp)：DNS・メールセキュリティ監視。DMARC/SPF/DKIM設定のドリフト検出とドメインセキュリティの可視化（SvelteKit + Cloudflare Workers）
- [courier.esolia.co.jp](https://courier.esolia.co.jp)：セキュアファイル共有。PIN保護と自動期限切れによる機密情報の安全な送受信（SvelteKit + Cloudflare Workers）
